# Priyanshu Chandra — Portfolio

A modern, responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript. Features a light/dark theme toggle, scroll-triggered animations via GSAP, and a clean card-based layout to showcase skills, projects, and services.

## Features

- **Dual Theme** — Seamless light/dark mode toggle with persistent visual preference
- **Scroll Animations** — GSAP ScrollTrigger drives smooth fade, slide, and scale-in effects across all sections
- **Responsive Design** — Fully adaptive layout across mobile, tablet, laptop, and desktop (5 breakpoints)
- **Animated Typing Header** — Role titles cycle through Front-end, Back-end, and UI/UX Design
- **6 Showcased Projects** — Each with GitHub and live demo links
- **Service Cards** — Web Designing, Model Designing (Blender), Game Designing
- **Mobile Hamburger Menu** — Collapsible navigation with animated icon

## Built With

| Technology | Purpose |
|---|---|
| HTML5 | Structure |
| CSS3 | Styling & themes |
| JavaScript (vanilla) | Interactivity, theme toggle |
| [GSAP](https://gsap.com/) 3.12 | Scroll-triggered animations |
| [Google Fonts](https://fonts.google.com/) | Poppins (300/400/500/600) |

No build tools, bundlers, or package managers — open `copytest.html` directly in a browser.

## Color Schemes

### Dark Theme

| Role | Color |
|---|---|
| Page background | `#0b0b12` |
| Card surface | `#1c1c2e` |
| Elevated surface | `#252540` |
| Primary text | `#eaeaf0` |
| Secondary text | `#a8a8c0` |
| Accent | `#6c8cff` |

### Light Theme

| Role | Color |
|---|---|
| Page background | `#ffffff` |
| Card surface | `#fafafa` |
| Primary text | rgb(85, 85, 85) |
| Accent | Orange (`rgb(255, 111, 1)`) |

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/priyanshu112c/your-repo-name.git
   ```
2. Open `copytest.html` in your browser — no server required.

Or deploy to any static host (GitHub Pages, Netlify, Vercel) by pointing it at the root directory.
